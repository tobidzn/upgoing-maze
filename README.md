# Upgoing Maze – Endless Reaction Game

## Overview

Upgoing Maze is a 2D endless game where the player controls a ball while navigating through incoming maze patterns. Instead of moving through a static level, the environment continuously moves toward the player, creating a constant need for quick reactions and precise movement.

The goal is to survive as long as possible by avoiding walls and finding safe paths through the maze.

---

## Core Mechanics

### Endless Maze System

The game uses a set of predefined maze templates that spawn continuously as the previous ones move off screen. This creates an infinite gameplay loop without repeating a fixed level.

### Movement and Control

The player controls a ball positioned in the center of the screen and must react to incoming obstacles in real time.

### Difficulty System

Each difficulty level changes how the game is played, not just how fast it is:

Easy allows full movement and includes teleportation on the sides of the screen.

Medium removes teleportation, requiring more precise positioning.

Hard restricts movement to left and right only, increasing the challenge significantly and forcing better timing.

---

## What I Learned

This project helped me understand how to create endless gameplay systems and manage spawning logic. I also learned how changing player constraints can drastically affect gameplay difficulty without modifying the core mechanics.

---

## Challenges

One of the main challenges was making the maze transitions feel smooth and consistent. Another was balancing the difficulty levels so each one felt distinct without breaking the core gameplay.
